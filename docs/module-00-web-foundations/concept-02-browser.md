# What is a Browser?

A browser is software installed on a computer or mobile device that requests, downloads, interprets, and displays websites and web applications.

Examples:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari

---

# Browser vs Search Engine vs Website

## Browser

Software used to access and display websites.

Examples:

- Chrome
- Edge
- Firefox
- Safari

---

## Search Engine

A website that helps users find other websites.

Examples:

- Google Search
- Bing
- DuckDuckGo

---

## Website

A collection of web pages hosted on a web server.

Examples:

- YouTube
- GitHub
- Amazon
- ChatGPT

---

# Four Responsibilities of a Browser

## 1. Networking

Communicates with web servers by sending HTTP requests and receiving responses.

---

## 2. Rendering

Converts HTML and CSS into the visual webpage displayed on the screen.

---

## 3. Running JavaScript

Executes JavaScript code to create interactive web applications.

---

## 4. Security

Protects users through HTTPS, permission management, sandboxing, and other security features.

User

# Real-Life Analogy

A browser is like a television.

The TV does not contain Netflix movies.

It requests and displays content from Netflix's servers.

Similarly, a browser requests and displays content from web servers.

- YouTube is a website.
- Browsers execute JavaScript.
